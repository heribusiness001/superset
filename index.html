1. SQL Lab: The Heart of Data Exploration üß™

The SQL Lab is your primary tool for interacting with raw data. It allows you to write, execute, and save SQL queries directly against your connected databases.

Mock SQL Lab Interface

// Select Database & Schema

📁 sales_db

📁 public

📊 transactions

📊 users

// Write your query

SELECT country, SUM(sales) AS total_sales

FROM transactions

GROUP BY country

ORDER BY total_sales DESC;

After running a query, click the **Explore** button to instantly send the results to the Chart Explore view, bypassing the need to manually create a Dataset.

In a Dataset, you define your **Metrics** (numerical values to aggregate, like `SUM(sales)`) and **Dimensions** (categorical values to group by, like `country`). Getting this right is crucial for easy chart creation.

6. User Access & Security: Controlling the Flow üîë

Superset's role-based access control system is crucial for managing who can see and do what within the platform.

Core User Roles

Role Key Permissions
Admin Full access to all features, including managing users, databases, and site-wide settings.
Alpha Can create and edit their own charts and dashboards. Can access all data sources.
Gamma Can view charts and dashboards they have been given access to. Cannot create or edit.
Public Unauthenticated users. Can only view dashboards specifically marked as public.

You can also create custom roles and define fine-grained permissions, such as granting access to specific datasets or dashboards.
